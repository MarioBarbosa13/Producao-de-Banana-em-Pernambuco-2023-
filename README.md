# Produção de Banana em Pernambuco (2023)

Este projeto analisou a produção de banana em alguns municípios que integram o Agreste Setentrional e Zona da Mata Norte de Pernambuco em 2023.  
Os dados foram visualizados através de gráficos e mapas choropleth.
